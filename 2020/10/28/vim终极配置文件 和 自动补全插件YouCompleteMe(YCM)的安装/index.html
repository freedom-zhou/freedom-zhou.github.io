运行编译脚本报错：

按理说我已经把需要的依赖都clone下来了，不应该再说我forgot to run:xxxx了，然而我跑到报错消息说的路径YouCompleteMe/third_party/ycmd/third_party/cregex下一看，发现只有个.git版本控制文件，的确是空的，于是我去码云上找到对应的路径看是否是空的，结果发现下面虽然有内容，但只是一个链接，链接到另一个仓库,而且还是github上的，真相大白：原来子模块下面还有子模块，用git clone命令只能下载下来当前模块，而子模块又得另外git clone
看来这种方法行不通，于是我准备在码云上找整个YouCompleteMe的所有代码并且包括依赖的整合仓库，这样就不用分开clone了，结果被我找到这个仓库：

它的备注是： 全码云下载 git clone --recurse https://gitee.com/haic2002/YouCompleteMe.git
这时候我才知道git clone有个--recurse参数，我估计这个参数可以递归的克隆该仓库下的子模块，点进去看它子模块的链接仍然是在码云上的，这样的话安装不就简单很多了吗，试试：

一开始还是挺顺利的，但安装到某个子模块的时候，告诉我没有权限clone该模块，这个子模块很可能是私有的，所以我无法下载。
好在还有另一个仓库，他把整个YouCompleteMe插件的所有文件都打包成压缩包放在服务器上了，直接wget下载压缩包就好了，仓库里只有一个README说明，该仓库的克隆还挺多，贴其中一个：

下载地址

下载了整合包，解压后运行python3 install.py,这次终于编译成功了

常见错误：
  1. 存在未安装的依赖，参见第四步，安装所有依赖
  2. vim版本低于7.4.1578或者不支持python2或python3，输入 git clone https://gitee.com/freedomzhou/vimrc.git ~/Desktop/vimrc
    cd ~/.vim/bundle
    tar -zxf ~/Desktop/vimrc/YouCompleteMe.tar.gz
