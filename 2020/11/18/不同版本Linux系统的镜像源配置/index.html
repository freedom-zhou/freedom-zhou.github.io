不同版本Linux系统的镜像源配置

Share

不同版本Linux系统的镜像源配置——解决软件包安装慢的问题

Ubuntu20 下配置镜像源

Ubuntu 安装软件的工具为apt
镜像源列表配置文件路径：/etc/apt/sources.list

亲测好用的镜像源
中国科学技术大学：
http://mirrors.ustc.edu.cn/ubuntu

若为带有图像界面的Ubuntu，可根据以下步骤自动选择最佳镜像源：
Software & Updates -> Ubuntu Software -> Download from: (点击下拉列表) -> Other ->Select Best Server
等待一段时间后，点击Choose Server，Ubuntu就自动配置好了当前访问速度最快的镜像源

Deepin 下配置镜像源

Deepin 安装软件的工具为apt
镜像源列表配置文件路径：/etc/apt/sources.list

Deepin15系统设置支持换源

Deepin20系统设置不支持换源了，因为换源可能会造成各种问题，所以官方去掉了这个功能，实在要换只能自己修改sources.list`文件

CentOS 下配置镜像源

CenOS 安装软件的工具为yum
镜像源列表配置文件路径：/etc/yum.repos.d/CentOS-Base.repo

亲测好用：
https://mirrors.ustc.edu.cn/centos/
中国科技大学镜像源 使用帮助：
http://mirrors.ustc.edu.cn/help/centos.html
